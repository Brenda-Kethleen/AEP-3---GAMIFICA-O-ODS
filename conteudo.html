Fase 1: HTML e CSS

HTML (HyperText Markup Language)

O que é HTML?

HTML é a linguagem de marcação usada para criar a estrutura de uma página web. Ele utiliza "tags" para definir diferentes partes do conteúdo, como títulos, parágrafos, links, imagens e outros elementos.

Estrutura Básica de um Documento HTML

Explicação das Tags:
!DOCTYPE html: Declara o tipo de documento como HTML5.

html lang="pt-BR": Inicia o documento HTML e define o idioma como português do Brasil.

head: Contém metadados, como o título da página e links para CSS.

meta charset="UTF-8": Define a codificação de caracteres como UTF-8.

meta name="viewport" content="width=device-width, initial-scale=1.0">: Faz com que a página seja responsiva em dispositivos móveis.

title: Define o título da página (aparece na aba do navegador).

body: Contém o conteúdo visível da página.

h1: Define um título principal.

p: Define um parágrafo.

a href="URL": Cria um link

CSS (Cascading Style Sheets)

O que é CSS?

CSS é a linguagem utilizada para descrever a aparência e a formatação de um documento HTML. Enquanto o HTML cuida da estrutura e do conteúdo de uma página web, o CSS cuida de como essa página deve ser apresentada visualmente. Com CSS, você pode definir cores, fontes, espaçamentos, alinhamentos, e muitos outros aspectos visuais dos elementos HTML.

Como Incluir CSS em uma Página HTML

Existem três maneiras principais de adicionar CSS a um documento HTML:
1- Inline CSS: Adiciona estilos diretamente nos elementos HTML. É útil para estilos rápidos e específicos.

2- Internal CSS: Adiciona estilos dentro da tag -style no cabeçalho -head do documento -HTML. É útil para páginas únicas.


3- External CSS: Linka um arquivo CSS externo ao documento HTML. É a forma mais comum e recomendada para manter o CSS separado do HTML.

Sintaxe Básica do CSS

A sintaxe do CSS é composta por seletores e declarações.
Seletores: Identificam quais elementos HTML serão estilizados.
Declarações: Descrevem o estilo que deve ser aplicado aos elementos selecionados. Elas são compostas por propriedades e valores.
